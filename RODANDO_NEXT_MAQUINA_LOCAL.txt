1. Clone e instalação

- Clonar e dar npm install em ambos os repos

2. Mudar a URL (front)

-     baseURL: "http://localhost:8080" no api.js (services)

3. npm start (back)

- npm start no repo do backend (cmd na pasta)

4. npx next build (front)

- no repo do front (cmd na pasta), buildar o next local pra compilar, toda vez que for mexer e salvar o codigo

5. npx next start (front)

- fazer isso no repo do front (cmd na pasta)

6. Acessar o localhost:3000 pra ver o site
